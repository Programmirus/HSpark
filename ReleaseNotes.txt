Релиз от 2021-05-10.

Состав релиза.
1. Реализовано расширение HSession класса pyspark.sql.session.SparkSession функциями для быстрой работы с партиционированными таблицами в HDFS:
Добавлены новые методы
  -статические методы
     Список партиций=get_table_partition(id спарк-сессии, имя таблицы)-получение списка партиций по имени таблицы.
     SparkDataFrame=get_partition_frame(id спарк-сессии, имя таблицы, партиция)-получение DataFrame заданной таблицы, отфильтрованной по заданной партиции.
  -Декоратор параллельного выполнения заданного пайплайна на DataFrame API по заданному списку партиций - parallelize_partition
  -Переопределен метод table базового класса для захвата только заданной партиции входной таблицы.

2.Реализована поддержка хранения пользовательских функций SparkUDF в HDFS с возможностью экспорта/импорта функций из HDFS в sparkSQL активной spark-сессии пользователями БД 
  и возможностью версионирования изменений.
   2.1.Реализована поддержка использования пользователями БД приватных sandbox в sqlite для добавления/чтения/редактирования SparkUDF разрабатываемых функций.
       Добавлены методы 
       Spark_alias=import_udf(имя функции, версия, sql_alias)-импорт заданной функции из sandbox
       register_udf(имя функции, описаниеб список тестов)-добавление новой функции в sandbox
   2.2. Реализована поддержка экспорта/импорта функций из sandbox в публичный UDFStorage БД HDFS.
        Добавлены методы 
        publish_udf(имя функции, версия, описание)-экспорт заданной функции из sandbox в UDFStorage
        Spark_alias=import_udf(имя функции, имя пользователя-создателя, версия, sql_alias)-импорт заданной функции из UDFStorage
   2.3. Реализован контроль ошибок выполнения сохраняемых функций.
        Добавлен статический метод CheckoutUDF(функция, список тестов) для контроля ошибок при вывзове функции на заданном списке тестов
